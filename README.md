# Manual Testing Practice (QA)

This repository contains manual QA practice materials including structured test cases, bug reports, and exploratory testing notes created for learning and demonstration purposes.

## What’s included
- **Test cases** written in a clear, repeatable format
- **Bug reports** with reproduction steps, expected vs actual results, and severity/priority
- **Exploratory testing notes** documenting findings and observations

## Tools
- Manual testing
- Chrome DevTools (basic)
- Markdown documentation

## Target role
Entry-level / Junior Manual QA Tester
